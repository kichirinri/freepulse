只有點讚，沒有評論
Scribe的所有文章只開放點讚功能，不設評論區。我們相信，每個人的思考與感悟應當保持獨立，不被他人的言論左右。
🌍
自由發表，以法律為底線
我們支持言論自由。但自由不是無邊界的——請在遵守所在地法律法規的前提下發文，任何煽動仇恨、違法的內容將被即時移除。
🤝
世界公民，懂得尊重他人
Scribe匯聚了來自47個國家和地區的華人。請以世界公民的胸懷，尊重與你不同的聲音。人身攻擊將永久封禁帳號。
